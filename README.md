## Подготовка  
В предлагаемый код необходимо внести ip адрес сервера, на котором поднят слушатель, и при необходимости изменить порт этого слушателя.  
По умолчанию порт слушателя прописан - 4444  

## Применение 
- Поднять на удаленном сервере слушателя \```nc -nlvp 4444 \```  
- Открыть мессенджер Экспресс  
- Нажать F12 для открытия среды разработки  
- Открыть вкладку Console (Консоль)  
- Если подобное взаимодействие делается впервые, то в консоли может стоять защита от вставки. Как правило, снимается вводом вразы "allow pasting", либо ей аналогичных  
- Вставить отредактированный под себя код  
- На слушателя тут же приходит внешнее подключение  

## Особенности  
- Представленная упрощенная версия мини-шелла работает, даже если заблокированы апплокером cmd и powershell. В условиях блокировки cmd + powershell энумерация директорий будет доступна, как и чтение содержимого.
- В условиях блокировки cmd + powershell не будет работать важная команда exec.
- Если описанных блокировок нет - этот скрипт можно самим доработать через ИИ, добавить возможности скриншотов, выгрузки/загрузки данных и тд.
- Скрипт работает, если EXpress есть в трее, даже с заблокированной сессией.
